JS Library Object

This code demonstrates basic OOP with Javascript. Interaction with this project is done through the console (F12, Cmd+Opt+I, Ctrl+Shift+I). The objects here are as follows:

  • Book()
  • Shelf()
  • Library()

These objects are accessed through the global 'lib'.

To get started we instantiate a Library Object. By default, the library will have a book capacity of 100 books and a shelf capactiy of 20. Otherwise we can pass the number of shelves and books it should have.

var lib1 = new lib.Library();
or
var lib1 = new lib.Library(20, 150);

Next we can initialize the library. This will fill the library object with the previously specified number of shelf and book objects, while also initializing those.

lib1.initAll();

After we initialize the library, the shelves will be empty. We can shelve all books in one go.

lib1.shelveAllBooks();

Otherwise we can put a book on a shelf or remove it by calling the enshelf() or unshelf() methods on an individual book object and passing the shelf in which the book should be placed.

lib1.books[2].enshelf(lib1.shelves[1]);
or 
lib1.books[2].unshelf(lib1.shelves[1]);

The Book Object

The book object has only a few typical properties: title, author, publish date, number of pages, and bound. These properties are all generated pseudorandomly. The titles are all generated by selecting combinations of words from different arrays or collectsion: nouns, adjectives, and roman numerals. Different couplings are then made with the words from each collection such as 'noun + noun', 'adjective + noun', 'The + adjective + noun', 'noun + roman_numeral', etc.

Publish date and page counts are generated simply by pseudo-randomly selecting a number within a range.

We can stringify/view all book titles quickly by calling a function on the library object literal. We can call methods to see all book bounds and all book years as well.

lib1.getAllBookTitles()
lib1.getAllBookBound()
lib1.getAllBookYears()
